# Small Python Projects

A collection of beginner-friendly Python projects to practice core programming concepts.

## 📁 Projects

### Temperature Converter 
Converts between Celsius and Fahrenheit.

### Food Delivery Discount System
A food delivery app gives discounts based on coupon codes and order amount.

### MAD-LIBS
Helps make a paragraph out of inputs .

### Username Validator 
A program which checks whether the username is valid according to the conditions. 

### CreditCardSlicer
Helps with credit card number slicing with help of different indexing methods.

### E-mail Slicer 
Helps in differentiating email with username and domain with help of indexing.
